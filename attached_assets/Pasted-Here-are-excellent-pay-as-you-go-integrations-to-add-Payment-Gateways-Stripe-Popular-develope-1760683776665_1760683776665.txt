	•	Square - Great for retail + appointments

Key Implementation Tip
For each integration, charge users based on:
	•	Per-transaction fees (payment gateways)
	•	Monthly API usage (SMS, email volumes)
	•	Connected account (flat fee per active integration)
	•	Tiered pricing (basic/pro features per integration)
You could also create integration bundles - e.g., “Marketing Bundle” (Mailchimp + Meta + Google Analytics) at a discounted rate.
